jgrep requirement doc

jgrep is obsolete, I use silver searcher now

like “grep –color”

  1. jgrep a single file

counterpart cmd:
grep -rn “verity” fs_mgr/fs_mgr.c

jgrep “verity” fs_mgr/fs_mgr.c
jgrep “verity” -f fs_mgr/fs_mgr.c

-f: specify a file[default]
